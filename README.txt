# Ksacas

Ksacas es mi primer proyecto web desarrollado por Marcos Palavecino para el curso frontend de Talento Tech, ksacas es una tienda en línea dedicada a la venta de ropa deportiva. Este proyecto web ha sido desarrollado utilizando HTML y CSS, y cuenta con un formulario de contacto funcional. Próximamente, se integrará JavaScript para mejorar la interactividad y funcionalidad del sitio.

## Características

- **Página de inicio**: Presenta una vista general del articulo mas vendido del mes con las caracteristicas del mismo.
- **Seccion productos**: Muestra una variedad de ropa deportiva con imágenes y una pequeña descripcion.
- **Seccion contacto**: Contiene un formulario funcional que permite a los usuarios enviar consultas y comentarios directamente desde la página de contacto.
- **Seccion sobre nosotros**: Contiene una breve descripcion acerca de Ksacas.
- **Diseño responsivo**: Adaptado para verse bien en dispositivos. -- (Mi problema es al querer que se vea bien en 320px el header me queda mal y no pude solucionar eso).


## Tecnologías utilizadas

- **HTML**: Estructura del contenido del sitio web.
- **CSS**: Estilos y diseño visual del sitio web.
